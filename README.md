# 5etools Homebrew – Shyverey

This repository contains homebrew JSON files for use with 5e.tools.

## Current Package

### Shyverey – Litover Black Escort

A near-TPK encounter featuring:

- Aventus, Inquisitor Commander
- Sister Mireal, Confessor
- Gardek Vayne, the Silent Shield
- Elias Rook, Heresy Hunter
- Sevast Kyle, the Lantern

Designed for:

- 5 level 9 PCs
- High-op tactical parties
- Story-heavy rival encounters
- Shyverey campaign setting

## Import into 5e.tools

### Step 1 – Import Homebrew

Open:

Utilities → Homebrew Manager → Import from URL

Use this raw file URL:

https://raw.githubusercontent.com/Heddin/5etools-homebrew/main/homebrew/shiverey-litover-black-escort-bestiary.json

### Step 2 – Use in Encounter Builder

After importing, the monsters will appear in Bestiary under source:

SHIVCOR / ShivCor

Search:

- Aventus
- Mireal
- Gardek
- Elias
- Sevast

Then add them manually into Encounter Builder.

## Notes

These are NOT player builds.

They are custom encounter monsters designed to emulate a coordinated anti-hero inquisitorial strike team.
